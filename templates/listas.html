Listas en HTML

Las listas en HTML son esquemas de organización para mostrar elementos o textos en orden listado en nuestra página web. Existen dos tipos de listas: Ordenadas por índice <ol></ol> (Ordered List), y sin orden por índice <ul></ul> (Unordered List).

Sea cual sea la lista que estemos utilizando, en todas las listas, se escriben los elementos con la etiqueta <li></li>, ingresando los datos a mostrar dentro de esta etiqueta.

Las listas ordenadas por índice son la mejor opción que podemos utilizar cuando queremos mostrar elementos ordenados con un número de índice que los identifique. Por ejemplo, si tuvieramos que listar los pasos que se deben realizar para crear un documento HTML, el código para lograrlo sería el siguiente:

<h3> Cómo inicializar un documento HTML5 </h3>
<ol> <li> Esribir la etiqueta "DOCTYPE", indicando que el tipo de documento es html. </li> <li> Iniciar el documento con la etiqueta "html". </li> <li> Ingresar la etiqueta "html", con los atributos de la página dentro de etiquetas de metadatos "meta". </li> <li> Agregar la etiqueta "body", y dentro de ella ingresar el contenido principal de la página. </li> </ol>

Cómo inicializar un documento HTML5

  1. Esribir la etiqueta "DOCTYPE", indicando que el tipo de documento es html.
  2. Iniciar el documento con la etiqueta "html".
  3. Ingresar la etiqueta "html", con los atributos de la página dentro de etiquetas de metadatos "meta".
  4. Agregar la etiqueta "body", y dentro de ella ingresar el contenido principal de la página.

Por supuesto las listas "ul" nos sirven demasiado, pues en ocasiones queremos listar datos sin importar el orden -índice- que estos lleven. Su estructura es exactamente la misma que utilizamos en las listas ordenadas. Ejemplo, para mostrar las etiquetas utilizadas:

<h3> Etiquetas utilizadas en listas HTML </h3>
<ul> <li> ul, ol </li> <li> li </li> <li> Podemos utilizar la etiqueta "a" para listar enlaces. </li> <li> Igual podemos listar textos con la etiqueta "p". </li> </ul>

Etiquetas utilizadas en listas HTML

  • ul, ol
  • li
  • Podemos utilizar la etiqueta a para listar enlaces.
  • Igual podemos listar textos con la etiqueta "p".

Conclusión

Las listas en HTML se pueden crear con las etiquetas <ol></ol> y <ul></ul>, y nos sirven para listar elementos ordenadamente y desordenadamente respectivamente.
