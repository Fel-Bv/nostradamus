Etiquetas Basicas en HTML.

Etiquetas de raiz.

Para todas las etiquetas es importante decir que se van a declarar con los signos de Menor que "<" y la etiqueta Mayor que "/>"

Las etiquetas de raiz, son aquellas que son necesarias para la elaboracion del mismo. La primera etiqueta es: La etiqueta doctype la cual expresamos de la siguiente manera <!DOCTYPE html/>.
La segunda etiqueta nos sirve para indicar donde se va a declarar el codigo en Html y se expresa de la siguiente manera <html> <html/>

Datos medios del documento.

En este caso los datos medios son aquellos que son requeridos para el cuerpo, como el titulo, la asosiacion con CSS y la mas importante que el head la cual vamos a expresar de la siguiente manera <head> <head/>


Tenemos tambien la etiqueta del titulo de nuestras pagina la cual es <title> <title/>
Una etiqueta importante es con la que podemos asociar una hoja donde le podemos dar estilos a nuestra pagina de html <title> <link/> dentro de esta vamos a declarar lo siguiente < rel="stylesheet" href="Declaramos el nombre de nuestra hoja de CSS">

Etiquetas de estructura HTML.

En este apartado tenemos las siguientes etiquetas: La etiqueta body en esta etiqueta es donde vamos a declarar y darle cuerpo a nuestra pagina <body> <body/>

La etiqueta main nos va a servir para definir el contenido princial de la pagina, cabe aclarar que solo podemos declarar una por documento <main> <main/>.

Dentro de estas etiquetas se contiene el contenido adicional de la pagina web, Suele ser contenido relacionado con la web pero de poca importancia. <aside> <aside/>.

Son etiquetas HTML de suma importancia con las cuales vamos a darles ese tamaño necesario, para diferenciar (titulos, subtitulos, etc) Estas etqieutas son <h1> <h1/>. <h2> <h2/>. <h3> <h3/>. <h4> <h4/>. <h5> <h5/>. <h6> <h6/>.

Etiqueta que nos va a servir para indicar el encabezado de una pagina: <header> <header/>.

Para el pie de la pagina tenemos la etiqueta <footer> <footer/>.

Etiquetas para la agrupacion de contenido.

La etiqueta para indicar la constancia de algun texto tenemos la etiqueta de parrafo <p> <p/>.

Tenemos una etiqueta para romper o espaciar entre textos esta es. <hr> <hr/>.

En este caso tenemos dos etiquetas que son utilizadas para la creacion de listas. La primera es <ol> <ol/>.
Esta nos va a servir para crear listas de manera ordenada, en caso contrario tenemos la etqueta: <ul> <ul/>.
Y tenemos una etiqueta que nos va a ayudar a recoger el contenido de una lista ya sea de manera, ordenada o des-ordenada, <li> <li/>.

Para crear algun contenedor o alguna caja tenemos la etiqueta <div> <div/>.

Etiquetas semanticas para el texto

Esta etiqueta es muy importante para crear hiper-enlaces, y es muy utilizada para los links. <a> <a/>.

Tenemos las etqieutas strong y small. La etiqueta strong nos va a ayudar para definir que una palabra es importante <strong> <strong/>. La etiqueta small va a dejar un comentario aparte como son los derechos de autor. <small> <small/>.

Esta etiqueta no tiene niguna funcion en comun asi que la vamos a utlizar en conjunto con los atributos, class y id .

La etiqueta br solo va a dar un salto de linea. <br> <br/>.

Etiquetas para la introduccion de contenido

Tenemos la etiqueta img, la cual nos va a insertar una imagen en nuestra pagina web. <img>.

Esta etiqueta nos va a ayudar a reproducir archivos de video, audio. <video> <video/>.

Con esta etiqueta vamos a introducir en la web un archivo de audio o stream de audio. <audio> <audio/>.
