Etiquetas Basicas en HTML.

Etiquetas de raiz.

Para todas las etiquetas es importabte decir que se van a declarar con los signos de Menor que "<" y la etiqueta Mayor que "/>"

Las etiquetas de raiz, son aquellas que son necesarias para la elaboracion del mismo. La primera etiqueta es: La etiqueta doctype la cual expresamos de la siguiente manera <!DOCTYPE html/> La segunda etiqueta nos sirve para indicar donde se va a declarar el codigo en Html y se expresa de la siguiente manera <html> <html/>

Datos medios del documento.

En este caso los datos medios son aquellos que son requeridos para el cuerpo, como el titulo, la asosiacion con CSS y la mas importante que el head la cual vamos a expresar de la siguiente manera <head> <head/> Tenemos tambien la etiqueta del titulo de nuestras pagina la cual es <title> <title/> Una etiqueta importante es con la que podemos asociar una hoja donde le podemos dar estilos a nuestra pagina de html <title> <link/> dentro de esta vamos a declarar lo siguiente < rel="stylesheet" href="Declaramos el nombre de nuestra hoja de CSS">
